Questions

The short answers.

Do I need a browser extension?

No. The Snap widget owns the right-click contextmenu event itself, so anyone on the page can file a bug without installing anything.

What gets attached to a report?

A full-page screenshot is captured automatically, plus the page URL and the surrounding console and network errors — grounded to the exact element you clicked.

Where do the bugs get filed?

Reports are auto-copied into your tracker via connectors for Plane, Jira, GitHub and Linear — or any endpoint via webhook.

Is Klavity Snap open source?

Yes. Klavity is open-core and self-hostable, so you can run the bug reporter on your own infrastructure.
